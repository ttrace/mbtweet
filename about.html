pbtweet app

pbtweet HTML5 Application is currentry alpha stage.

Install

Follow instruction on pbtweet application authentication page.
pbtweet app works on OAuth on Twitter API authentication.
pbtweet app runs on latest WebKit browsers (Safari 4 and Chrome 4 beta).
